9.3 #settings.gs Grid system

Settings that affect the work of grid system both modern and legacy.

  • modern grid is based on CSS Grid and is built using common grid utilities, such as Gtc, Gap etc
  • legacy grid system is based on flexbox and works with helpers and custom utilities, such as -Gscc, -Gsf etc.
Parameters:
  • $legacy-grid-system
    enable legacy grid system and generate some CSS for it
    Defaults to: false
  • $gs-default-gtr
    default gutter on large width screen. Used as fallback for --ml-gg.
    Defaults to: 4su
  • $gs-columns
    columns count.
    Defaults to: 12
  • $gs-gutters
    map with grid gutters on breakpoints.
