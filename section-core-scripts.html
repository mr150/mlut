6.1 #core-scripts.init Init

Toggle full screen Open in new window Toggle example guides Toggle HTML markup

Implements js and nojs states for utilities.

When loading init.js, it will remove js state from the utilities names and they will work as usual. Utilities with nojs state also will be removed from the markup when loading this script.

Example

Color white if js disabled

W9c on md if js enabled

Markup: ../docs/examples/core-scripts/init.html 
<style>
	.nojs_C-f {
		color: #fff;
	}
</style>

</div>
