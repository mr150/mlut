5 #utils-reference Utils reference

All utilities that available in the registry.

This reference is generated on the basis of settings in the code. In the left column, the options of the utility are specified, and in the right - their values. In maps with values, the default is the CSS value that will be used without specifying the utility value. Speaking more accurately, default is a ''(empty string) in the utility value:

@include ml.mk-util('D', '');

// CSS
.D {
	display: block;
}

After the options, the values of utility that are contained in the CSS library (AOT) may be specified. The values are written in Generation Syntax
