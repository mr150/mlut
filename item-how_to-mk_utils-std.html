3.1.1 #how_to.mk_utils.std Standard ways

Standard methods include creating through config and using mk- mixins. As the values of utilities, you can use a list or map. The components syntax can be used both in the names of utilities/groups and in the values.

@include ml.mk-util(
	'Mxw',
	(
		'cardXs': 12r,
		'sm0': 560px,
	)
);

// CSS
.Mxw-cardXs {
	max-width: 12rem;
}

.Mxw-sm0 {
	max-width: 560px;
}

If you add new values via map, they will appended to the registry and after ,you can use them again.

You can find other examples of the standard way usage in the basic concepts.
